Running the code
--------------------------------------------------------------------------------------------------
 - Run the main file using > python main.py

 - Some files will be outputted into folders
    - regression_files contains the output for the regression model
    - cluster_visualisations contains the output for the clustering model
    - cluster_evaluations contains data on the clustering model

 - Input Generation: Input your generation to get recommended books based on your generation
 Generations = [Silent Generation, Baby Boomers, Generation X, Millenials, Generation Z]
